# kbdタグを装飾するCSS

毎回ブログのテーマを変えるたびに消しちゃうのでメモっとく。

~~~ css
kbd {
    color: #333;
    border: solid 1px #999;
    border-width: 1px 2px 2px 1px;
    margin: 0 3px;
    padding: 1px 5px;
}
~~~
